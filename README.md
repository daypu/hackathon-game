# 西游 · 万象迷途

一个基于 Canvas 2D 的像素风西游题材游戏原型。

唐僧师徒在前往西天取经的途中误入由妖气幻化出的“万象迷途”。玩家操控取经队伍在固定视角的二维像素场景中自由移动，穿越长安营地、流沙河、火焰山、盘丝洞、万象妖林与灵山入口。每个场景相机固定不动，走到发光传送点后会直接切换到下一个场景。

玩家需要躲避妖风、落石、火焰山余焰、盘丝洞蛛网和小妖陷阱，同时收集佛光、经文残页、仙桃、筋斗云碎片和护身符，维持法力、信念与经书完整度。

## 运行

```bash
npm install
npm run dev
```

打开 `http://localhost:5173/` 即可游玩。

## 操作

- `WASD` / 方向键：自由移动取经队伍
- 走到发光传送点：切换场景
- 走到灵山宝塔金光处：结算取经成果
- `Space` / `Enter`：开始、确认、重新开始
- `P` / `Esc`：暂停或返回
- `M`：静音

## 当前版本

- 像素风菜单、固定视角场景探索与结算场景
- 唐僧、孙悟空、猪八戒、沙僧四人队列
- 三项核心属性：法力、信念、经书完整度
- 五类妖障：妖风、落石、火焰山余焰、盘丝洞蛛网、小妖陷阱
- 五类正向元素：佛光、经文残页、仙桃、筋斗云碎片、护身符
- 固定相机单屏场景、场景传送点、区域发现和场景舆图
- 区域妖气强度、取经探索进度、终点灵山
- 根据表现生成取经称号、评分与成就

## 开放世界方向

当前版本采用类似参考图的“单屏固定场景 + 定点切换”结构。后续可以继续扩展：

- 更大的大地图：加入女儿国、车迟国、通天河、狮驼岭等区域
- 更强的探索：多出口场景、支线路径、迷雾解锁和隐藏洞府
- 区域生态：不同地形、妖怪、法宝、事件和环境机制
- 队伍成长：角色技能、法宝升级、经书修复、信念系统
- 任务系统：主线取经、支线除妖、隐藏成就和区域 Boss
