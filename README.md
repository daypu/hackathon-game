# 西游 · 万象迷途

一个基于 Canvas 2D 的像素风西游题材游戏原型。

唐僧师徒在前往西天取经的途中误入由妖气幻化出的“万象迷途”。当前版本是一张融合大地图（关卡总图 / hub）：玩家操控孙悟空在地图上自由探索，相机随角色移动，左上角提供全局“万象舆图”小地图。大地图自西向东依次串联 **花果山 → 高老庄 → 流沙河 → 火焰山 → 灵山雷音寺** 五处地标，每处地标放置一个发光法阵，走入法阵即可进入对应关卡（关卡玩法内容开发中）。

## 运行

```bash
npm install
npm run dev
```

打开 `http://localhost:5173/` 即可游玩。

如需局域网内其他设备访问：

```bash
npm run dev -- --host
```

## 操作

- `WASD` / 方向键：自由移动（孙悟空）
- 走入发光法阵后按 `Space` / `Enter`：进入对应关卡
- `P` / `Esc`：暂停 / 继续
- `M`：静音

## 当前版本

- 像素风菜单与孙悟空登场动画（含奔跑动作）
- 单张融合大地图自由探索（`src/assets/maps/world.png`，2508 × 1410），相机跟随
- 左上角全局小地图“万象舆图”，标注法阵与当前视野
- 5 个关卡触发法阵：花果山、高老庄、流沙河、火焰山、灵山雷音寺
- 走入法阵显示提示，按确认键触发对应关卡（已预留统一接入接口）

## 关卡玩法（开发中）

进入法阵后会切换到对应关卡场景，目前各关卡的具体玩法尚未实现（触发后仅提示“开发中”）。以下素材已在 `src/config.js` 中定义，供后续关卡复用：

- 五类妖障：妖风、落石、火焰山余焰、盘丝洞蛛网、小妖陷阱
- 五类正向元素：佛光、经文残页、仙桃、筋斗云碎片、护身符
- 取经称号、评分与成就评价

接入位置见 `src/scenes/PlayScene.js` 中的 `#startLevel(lv)`，`lv.key` 取值：`huaguoshan` / `gaolaozhuang` / `liushahe` / `huoyanshan` / `leiyinsi`。

## 调试

- 在地址后追加 `?overview`（如 `http://localhost:5173/?overview`）：进入全图法阵校对视图，将整张大地图缩放到一屏，叠加显示 5 个法阵圈位置、坐标、出生点与角色位置。移除该参数即可恢复正常游玩。

## 项目结构

```
src/
├── main.js              # 入口
├── config.js            # 全局配置、属性 / 妖障 / 收集物 / 称号数据
├── openWorld.js         # 大地图、法阵定义、相机与绘制
├── engine/              # Game 主循环、Input、Renderer、audio、utils
├── entities/            # Player、Hazard、Pickup、particles
├── scenes/              # MenuScene、PlayScene（hub）、ResultScene
├── systems/             # Spawner、ScoreSystem
├── sprites/             # 孙悟空像素精灵与动画帧
├── ui/                  # HUD、小地图
└── assets/maps/         # world.png 融合大地图
```
