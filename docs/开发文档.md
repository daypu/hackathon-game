> **项目代号**：万象迷途（Journey Maze）
> **赛事**：抖音AI创变者计划黑客松
> **开发周期**：24小时

## 一、项目概述

### 1.1 游戏定位
一款基于西游记IP的**像素风2D跑酷游戏**，核心创新点是**双角色克制切换机制**——玩家需要根据前方障碍类型，在悟空和八戒之间快速切换，考验观察力和反应力。

### 1.2 核心卖点
- 🎯 **策略性切换**：不同障碍只能由对应角色处理
- ⚡ **即时反馈**：技能释放爽快，连击得分高
- 🎨 **像素风格**：复古怀旧，符合大众审美
- 📱 **社交传播**：操作方式醒目，结算可分享

### 1.3 目标体验
- **第一局**：30秒内死亡，但觉得"再来一次"
- **第十局**：能坚持2分钟，体验到爽快感
- **第百局**：开始挑战高分，主动分享

## 二、核心玩法设计

### 2.1 基础规则

**游戏目标**：操控唐僧师徒在火焰山持续前进，躲避障碍、收集奖励，坚持越久得分越高。

**游戏循环**：
```
开始 → 跑酷 → 遇障碍 → 切换角色/释放技能 → 收集奖励 
→ 难度增加 → 死亡 → 结算 → 重玩
```

**胜负判定**：
- 失败：用错误的角色碰到障碍物
- 无终点：无尽模式，挑战最高分

### 2.2 角色系统

#### 悟空（灵活型 🐒）
| 属性 | 数值 |
|------|------|
| 颜色 | 橙色 #FF6600 |
| 形状 | 方块 32x32 |
| 跳跃高度 | 1.2x（更高） |
| 移动速度 | 1.0x |
| 体积判定 | 小 |

**特殊能力**：
- ✅ **免疫火焰**：可以直接穿过地面火焰
- ✅ **金箍棒横扫**（技能）：清除屏幕内所有火焰类障碍
  - 冷却时间：10秒
  - 持续时间：1秒动画

**弱点**：
- ❌ 碰到岩石立即死亡
- ❌ 碰到飞石立即死亡

#### 八戒（力量型 🐷）
| 属性 | 数值 |
|------|------|
| 颜色 | 粉紫色 #FF69B4 |
| 形状 | 圆形 36x36 |
| 跳跃高度 | 1.0x |
| 移动速度 | 0.9x（稍慢） |
| 体积判定 | 大 |

**特殊能力**：
- ✅ **免疫岩石**：可以直接撞碎地面岩石
- ✅ **吸收范围x1.5**：金币吸附范围更广
- ✅ **变胖冲撞**（技能）：3秒无敌，撞碎所有岩石
  - 冷却时间：12秒
  - 持续时间：3秒

**弱点**：
- ❌ 碰到火焰立即死亡
- ❌ 体积大更容易碰到飞石

### 2.3 障碍物系统

| 障碍物 | 视觉 | 出现位置 | 处理方式 | 出现时机 |
|--------|------|---------|---------|---------|
| 🔴 火焰 | 红色三角 | 地面 | 悟空穿过 / 八戒跳过 | 0秒起 |
| 🪨 岩石 | 灰色方块 | 地面 | 八戒撞碎 / 悟空跳过 | 0秒起 |
| 🔥 火墙 | 红色全屏 | 全屏 | 必须悟空技能 | 30秒起 |
| 🗿 巨石 | 大灰球 | 滚动 | 必须八戒技能 | 45秒起 |
| 💨 飞石 | 灰色圆 | 空中 | 两者都需跳过 | 15秒起 |
| 🕸️ 蛛网 | 白色网 | 空中低位 | 两者都需蹲下/低跳 | 60秒起 |

### 2.4 奖励系统

| 奖励 | 视觉 | 分数 | 效果 |
|------|------|------|------|
| ⭐ 金币 | 黄色星 | +10 | 基础分数 |
| 📜 经文 | 蓝色书 | +50 | 高分奖励 |
| ☁️ 筋斗云 | 白云 | +20 | 短暂加速2秒 |
| 🍑 仙桃 | 桃形 | +30 | 恢复一次复活机会 |

### 2.5 计分机制

**基础分数**：
- 时间分：每秒 +5 分
- 收集分：根据奖励类型
- 距离分：每米 +1 分

**连击系统**：
- 连续收集5个奖励 → x1.5 倍数
- 连续收集10个奖励 → x2.0 倍数
- 碰到障碍/切换角色 → 重置连击

**完美技能**：
- 用技能恰好清除3个以上障碍 → +100分 + "完美！"提示

### 2.6 难度曲线

| 时间段 | 速度倍率 | 障碍间隔 | 新增障碍 |
|--------|---------|---------|---------|
| 0-15s | 1.0x | 2.0s | 火焰、岩石 |
| 15-30s | 1.1x | 1.8s | + 飞石 |
| 30-45s | 1.2x | 1.6s | + 火墙 |
| 45-60s | 1.3x | 1.4s | + 巨石 |
| 60-90s | 1.5x | 1.2s | + 蛛网 |
| 90s+ | 1.8x | 0.8s | 全障碍混合 |

## 三、交互设计

### 3.1 操作方式

**桌面端（开发优先）**：
- `空格 / ↑`：跳跃
- `1`：切换悟空
- `2`：切换八戒
- `J`：释放当前角色技能
- `↓`：蹲下（躲蛛网）
- `ESC`：暂停

**移动端（适配）**：
- 上滑：跳跃
- 左滑：切换悟空
- 右滑：切换八戒
- 点击屏幕：释放技能
- 下滑：蹲下

#### Wukong 类
```javascript
class Wukong extends Player {
    constructor(scene, x, y) {
        super(scene, x, y);
        this.jumpPower = 600;  // 跳更高
        this.color = 0xFF6600;
        this.skillName = '金箍棒横扫';
        this.skillMaxCD = 10000;
    }
    
    canPass(obstacle) {
        // 火焰：可以穿过
        if (obstacle.type === 'fire') return true;
        // 岩石/飞石：死亡
        if (['rock', 'fly_rock'].includes(obstacle.type)) return false;
        return null; // 让基类处理
    }
    
    useSkill() {
        if (this.skillCooldown > 0) return;
        // 清除屏幕内所有火焰类障碍
        this.scene.obstacles.getChildren()
            .filter(o => ['fire', 'fire_wall'].includes(o.type))
            .forEach(o => o.destroy());
        // 播放特效
        this.scene.effects.playWukongSkill();
        this.skillCooldown = this.skillMaxCD;
    }
}
```

#### Bajie 类
```javascript
class Bajie extends Player {
    constructor(scene, x, y) {
        super(scene, x, y);
        this.speed = 270;  // 稍慢
        this.bodyRadius = 18;  // 体积大
        this.absorbRange = 80;  // 吸收范围
        this.color = 0xFF69B4;
        this.skillName = '变胖冲撞';
        this.skillMaxCD = 12000;
    }
    
    canPass(obstacle) {
        if (obstacle.type === 'rock') return true;
        if (['fire', 'fire_wall'].includes(obstacle.type)) return false;
        return null;
    }
    
    useSkill() {
        if (this.skillCooldown > 0) return;
        this.invincible = true;
        this.scene.time.delayedCall(3000, () => {
            this.invincible = false;
        });
        this.skillCooldown = this.skillMaxCD;
    }
}
```

#### ObstacleManager
```javascript
class ObstacleManager {
    constructor(scene) {
        this.scene = scene;
        this.baseInterval = 2000;
        this.lastSpawnTime = 0;
        this.availableTypes = ['fire', 'rock'];
        this.gameStartTime = scene.time.now;
    }
    
    update(time, delta) {
        const elapsed = time - this.gameStartTime;
        
        // 根据时间增加障碍类型
        if (elapsed > 15000 && !this.availableTypes.includes('fly_rock')) {
            this.availableTypes.push('fly_rock');
        }
        if (elapsed > 30000 && !this.availableTypes.includes('fire_wall')) {
            this.availableTypes.push('fire_wall');
        }
        // ... 更多类型
        
        // 计算生成间隔
        const interval = Math.max(800, this.baseInterval - elapsed * 0.01);
        
        if (time - this.lastSpawnTime > interval) {
            this.spawnObstacle();
            this.lastSpawnTime = time;
        }
    }
    
    spawnObstacle() {
        const type = Phaser.Math.RND.pick(this.availableTypes);
        new Obstacle(this.scene, type);
    }
}
```

## 九、开发规范

- **所有文档**（`.md`、`.txt`、设计图）→ 必须放在 `docs/`

- 类名：PascalCase（如 `GameScene`）
- 方法/变量：camelCase（如 `useSkill`）
- 常量：UPPER_SNAKE_CASE（如 `MAX_SPEED`）
- 文件名：与主要类同名

### 代码规范
- 每个文件不超过200行
- 关键逻辑必须加注释
- 配置参数放config.js
- 使用ES6+语法

### Git规范（如果使用）
- commit信息：`[阶段X] 简短描述`
- 每完成一个小功能就commit
- 重要节点打tag
