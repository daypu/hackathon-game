# 流沙河：木筏惊魂（独立版）

一个可独立运行的 H5 Canvas 小游戏（Vite + 原生 Canvas 2D），打开后直接进入开始界面。

## 运行

```bash
npm install
npm run dev
```

然后打开终端输出的本地地址（通常是 `http://localhost:5173/`）。

## 目标

撑过 30 秒，不让木筏翻！

## 操作

- `A/D` 或 `←/→`：左右稳船（核心）
- `Space` / `Enter`：安抚唐僧
- `1`：修理木筏
- `2`：击退河怪
- `P` / `Esc`：暂停 / 返回开始界面

## 成功 / 失败

- 成功：撑过 30 秒
- 失败：倾斜超过 ±40° 太久（翻船）或木筏耐久归零（破碎）

成功或失败后：

- `Space` / `Enter`：重新挑战
- `Esc`：返回开始界面
